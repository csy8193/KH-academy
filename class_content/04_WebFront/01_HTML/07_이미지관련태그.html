이미지 관련 태그

img 태그 : HTML문서 (== 웹문서, 웹페이지)에 사진, 그림을 삽입할 때 사용하는 태그

        img 태그 속성
        1) src : 삽입할 이미지 경로를 지정하는 속성(컴퓨터 내부 경로, url)
        2) width : 이미지의 너비(폭, 가로길이)를 지정하는 속성
        3) height : 이미지의 높이(세로 길이)를 지정하는 속성
        4) alt : 이미지를 설명하는 속성. 이미지가 출력되지 않는 경우 화면에 표시됨

img 태그 활용하기

img 태그에 width, height 중 하나의 크기만을 지정하는 경우
지정되지 않은 방향의 크기도 같은 비율로 조정되게 된다.

width, height를 모두 지정하는 경우 이미지 비율을 무시하고 지정된 크기로 화면에 출력된다.

alt 속성 확인하기

1. 이미지의 크기가 화면을 벗어나는 경우
1) 이미지 1장인 경우 : 좌우 스크롤바 생성
2) 이미지 여러장인 경우 : 화면 크기를 벗어난 이미지를 다음 줄로 이동

2. img태그는 inline(화면을 수평 분할) 형식의 태그이다.
-> inline 형식 태그는 기본적으로 크기 지정이 불가능하나,
이미지 또는 멀티미디어 관련 태그는 크기 지정이 가능하다

고정 크기 단위, 가변 크기 단위

1) 고정 크기 단위 : 화면 크기에 관계 없이 지정된 값 만큼의 크기가 지정되는 단위
(px : 화면 내부에 색을 표현하는 작은 전구 하나를 1px)

2) 가변 크기 단위 : 화면 또는 지정된 영역의 기준 사이즈에 따라 변경되는 단위
( % : 부모 요소로부터 지정된 퍼센티지 만큼의 크기를 지정 )
