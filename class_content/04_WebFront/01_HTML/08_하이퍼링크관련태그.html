하이퍼링크 관련 태그

        하이퍼링크는 웹 문서가 일반적인 문서와 구분되는 가장 핵심적인 기능이다.
        클릭을 통해 연결된 다른 문서로 이동하는 방법을 편리하게 제공하는 기능

        1) 텍스트를 클릭해서 다른 문서로 이동
        2) 이미지를 클릭해서 다른 문서로 이동
        3) 같은 페이지 내에서 이동하는 방법

a 태그(anchor : 닻, 고정시키다, 묶어두다)

href 속성 (hypertext reference) : 이동할 페이지를 작성하는 속성

  • 글자 관련 태그
  • 목록 관련 태그
  • 글자, 목록 태그 연습문제

img 태그를 이용한 링크 만들기

title 속성 : 링크에 마우스 오버를 하는 경우 설명을 나타내는 속성

학원

target 속성 : a태그 클릭 시 페이지가 열리는 방법을 지정

        _self : 현재 창에서 지정된 링크로 이동(기본값)
        글자 관련 태그

        _blank : 새 창에서 지정된 링크로 이동
        목록 관련 태그

        _parent : 부모 창에서 열기
        _top : 가장 상위 창에서 열기

한 페이지 내에서 이동(점프)하는 a태그 만들기
