remove(), empty(), detach()

        remove() : DOM 요소 잘라내기 // 화면에서 제거하면서 해당 요소를 반환
                                     // -> 요소만 잘라낼 뿐 이벤트는 유지되지 않는다

        detach() : DOM 요소 떼어내기 // 화면에서 제거하면서 해당 요소를 반환
                                     // -> 요소 뿐만 아니라 이벤트도 유지된다
        (detach : 떼어 --- attach : 붙이다)

        empty() : 자식 요소를 제거 // innerHTML = "", html("") 대체
안녕

clone([true | false])

        요소 자체를 복제
        매개변수가 true인 경우 : 이벤트도 복제 O
        매개변수가 false인 경우 : 이벤트도 복제 X (default)
안녕

each() 메소드

        jQuery 방식의 향상된 for문
        (향상된 for문 == for each문)

        1. 배열 또는 다수의 요소를 순차 접근 하는 방식
        $.each( object, function([index, item]){})
            - object : 배열(자료형이 object)
                       다수의 요소(HTML DOM 객체)

            - function(){} : 반복 수행할 코드

            - 매개변수 index : 현재 접근중인 index 표시
            - 매개변수 item : 현재 접근중인 요소

            - function(){} 내부 this : 현재 접근중인 요소
        
        

        2. 다수의 요소를 순차 접근 하는 방식
        $("선택자").each( function( [index, item] ){} );
